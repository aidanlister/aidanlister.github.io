Here's some of my projects:

PyXero

PyXero is the official library for connecting to Xero with Python. It's primarily maintained by myself, and Russel Keith Magee of the Django foundation.

» See the project on github.

Python for Facebook

I took over maintenance of the official python library for the facebook API A home for Python developers working with Facebook. We maintain two repositories on github:

facebook-sdk is a set of essential tools for working with Facebook in Python,

django-facebook is an extensible django plugin for building facebook integrated sites.

» See the project on github.

Drupal

Drupal is an open source content management platform powering millions of websites and applications. I've authored over 20 modules that are widely used throughout the Drupal ecosystem, collectively used across thousands of sites and registering over 120,000 downloads.

» See my projects on Drupal.org.

PEAR

The PHP Extension and Application Repository (PEAR) was a central part of the PHP ecosystem before frameworks arrived on the scene. I authored several packages which were collectively downloaded over 350,000 times and powered many open source applications like Wordpress, Joomla, Drupal, and Wikipedia.

» See my projects on Drupal.org.

PHP.net

I was a noted author of the PHP documentation, and a notes maintainer for several years. I started using PHP3 just prior to PHP4's release, and contributed extensively to the PHP5 documentation when PHP5 was released.

» See the attribution and my favourite manual page.

Code repository

I maintain a repository for useful functions and classes, some of which have ended up in open source frameworks which power a sizeable chunk of the worlds websites, including Wordpress, Drupal, and Joomla.
